# Extension back-to-top-button

A "Back to top" button is displayed on all CMS pages.

## Configuration

* `buttonWhitelist` - (Array) Desired array of pathname patterns given as strings.

## Default buttonWhitelist value
```json
{
  "buttonWhitelist": [
    "/page/:pageId"
  ]
}
```
